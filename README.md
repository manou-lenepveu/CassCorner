# Kiera Cass Books Website

This project is a website dedicated to the author Kiera Cass.  
It presents her published books and allows visitors to explore and purchase them online.

## Features

- Author presentation
- List of published books
- Book details pages
- Online purchase option
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose of the Project

The goal of this project is to create a modern website that showcases the works of Kiera Cass and provides readers with an easy way to discover and buy her books.
